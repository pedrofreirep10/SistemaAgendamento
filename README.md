# Sistema de Agendamento

Projeto de um sistema de gerenciamento de compromissos desenvolvido em Python utilizando Programação Orientada a Objetos.

O sistema permite cadastrar, visualizar e organizar compromissos, além de verificar conflitos de horários e facilitar o controle de uma agenda.

## Funcionalidades

- Cadastrar compromissos com título, data, horário e duração
- Listar compromissos cadastrados
- Buscar compromissos por data
- Visualizar agenda diária e semanal
- Alterar e remover compromissos
- Verificar conflitos de horário
- Validar dados de entrada
- Salvar e carregar compromissos
